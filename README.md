# Ali Flour Mill Management System

A comprehensive web-based management system for Ali Flour Mill with stock management, grinding calculations, and cash accounting.

## Features

- **📦 Stock Management**: Track inventory of all flour mill products
- **⚙️ Grinding Calculator**: Automatic calculation of grinding percentages
- **💰 Cash Accounting**: Income and expense tracking with balance calculation

## Products Tracked

- Super 20kg
- Super 10kg
- Aata 20kg
- Aata 10kg
- F 20kg (Maida)
- Meda 50kg
- Suji 50kg
- Chokar 20kg
- Chokar 49kg

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Usage

### Stock Management
- Add or remove bags from inventory
- View total stock count
- Data persists in browser storage

### Grinding Calculator
- Enter number of bags for each product
- Calculate total grinding weight
- View percentage breakdown for each product

### Cash Accounting
- Record income and expense transactions
- View daily balance
- Track transaction history

## Data Storage

All data is stored locally in your browser using localStorage. No server required!
